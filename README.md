# Delivery App (Etapa 2)

Aplicación de gestión de logística y entregas para el módulo de Delivery, construida con Next.js, Prisma y Supabase.

## Accesos
- **URL Producción:** [Pendiente de Deploy]
- **Usuario Admin:** [Pendiente de Auth]
- **Contraseña Admin:** [Pendiente de Auth]

## Descripción
Esta webapp funciona de forma independiente y maneja el ciclo de vida de una entrega (asignación, tracking, cambio de estado). Expone su propia API y utiliza una base de datos PostgreSQL aislada.
